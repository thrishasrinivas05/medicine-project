# Medicine Project 

This is a Flask-based web application for medicine-related functionality.

## Features
- User login system
- Dashboard interface
- OCR and ML-based processing

## Technologies Used
- Python (Flask)
- HTML, CSS
- Machine Learning
- OCR

## How to Run Locally

1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python app.py
